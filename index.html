Welcome!

This is a beginner-friendly guide to start using Git and revisit HTML & CSS skills. Let's keep it simple and clean, with a touch of Thai style.

HTML Essentials

  • <header>, <nav>, <main>, <footer> for page structure.
  • Use semantic tags (<article>, <section>, <aside>).
  • Add alt text to images for accessibility.
  • Headings: <h1> to <h6> in order.
  • Links: <a href="url">text</a> for navigation.

CSS Essentials

  • Responsive design via flex & grid.
  • Use rem or em for font sizes.
  • Maintain sufficient color contrast for readability.
  • Utility-first classes (Tailwind) speed up development.
  • Mobile-first: style small screens first, then add breakpoints.
